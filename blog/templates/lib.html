Инструменты для обработки спутниковых изображений

Существует множество инструментов и программного обеспечения, которые можно использовать для обработки спутниковых изображений. Вот некоторые из них:
1. Общедоступные программы:
   - Google Earth Engine: Это облачная платформа от Google, предназначенная для анализа и обработки огромного объема спутниковых изображений.
   - QGIS: Бесплатная и открытая геоинформационная система, которая поддерживает обработку спутниковых изображений и множество геоаналитических возможностей.
   - GRASS GIS: Еще один бесплатный и открытый геоинформационный инструмент, включающий мощные возможности обработки спутниковых данных.
2. Коммерческое программное обеспечение:
   - ENVI: Это программное обеспечение для анализа и обработки спутниковых изображений, который предлагает различные функции, такие как классификация, композиты изображений и оценка качества данных.
   - ERDAS IMAGINE: Это популярное программное обеспечение для обработки спутниковых изображений, предлагающее множество инструментов для анализа и обработки геопространственных данных.
3. Открытые библиотеки и пакеты:
   - GDAL (Geospatial Data Abstraction Library): Открытая библиотека, предоставляющая набор инструментов для чтения, записи и обработки спутниковых изображений и других геоинформационных данных.
   - Rasterio: Python-библиотека, которая предоставляет инструменты для чтения и записи спутниковых изображений в формате Raster и их обработки.
   - OpenCV: Библиотека компьютерного зрения и обработки изображений, которая может быть использована для обработки спутниковых изображений.
